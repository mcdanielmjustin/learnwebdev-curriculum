Flexbox vs Grid

Flexbox is great for one-dimensional layouts (rows OR columns). Grid is designed for two-dimensional layouts (rows AND columns simultaneously).

Use Grid when you need:

  • Complex page layouts (header, sidebar, main, footer)
  • Card grids that align both horizontally and vertically
  • Any layout where you think in terms of rows AND columns

Creating a Grid

grid-basics.css 
.container {
  display: grid;
  grid-template-columns: 200px 200px 200px;  /* 3 columns, each 200px */
  grid-template-rows: 100px 100px;         /* 2 rows, each 100px */
  gap: 16px;                                 /* Space between cells */
}

The fr Unit (Fractional)

The fr unit represents a fraction of available space. It's like flex-grow for Grid!

.container {
  display: grid;
  grid-template-columns: 1fr 1fr 1fr;    /* 3 equal columns */
  grid-template-columns: 1fr 2fr 1fr;    /* Middle is twice as wide */
  grid-template-columns: 250px 1fr;      /* Fixed sidebar + flexible main */
}

repeat() Function

Instead of typing the same value multiple times, use repeat():

/* These are equivalent */
grid-template-columns: 1fr 1fr 1fr 1fr;
grid-template-columns: repeat(4, 1fr);

/* Responsive grid: as many columns as fit, minimum 250px each */
grid-template-columns: repeat(auto-fit, minmax(250px, 1fr));
The Magic Line

grid-template-columns: repeat(auto-fit, minmax(250px, 1fr)) creates a responsive grid that automatically adjusts columns based on screen width. No media queries needed!

Placing Items

Grid items can span multiple rows or columns:

grid-placement.css 
.item {
  grid-column: 1 / 3;      /* Start at line 1, end at line 3 (span 2 columns) */
  grid-column: span 2;     /* Span 2 columns from wherever it lands */
  grid-row: 1 / 4;         /* Span 3 rows */
}

/* Shorthand: grid-area: row-start / col-start / row-end / col-end */
.header {
  grid-area: 1 / 1 / 2 / 4;  /* First row, span all 3 columns */
}

Named Grid Areas

For complex layouts, name your areas for clarity:

dashboard-layout.css 
.dashboard {
  display: grid;
  grid-template-columns: 250px 1fr;
  grid-template-rows: 64px 1fr;
  grid-template-areas:
    "header header"
    "sidebar main";
  min-height: 100vh;
}

.header  { grid-area: header; }
.sidebar { grid-area: sidebar; }
.main    { grid-area: main; }

Common Grid Patterns

Card Grid

.card-grid {
  display: grid;
  grid-template-columns: repeat(auto-fill, minmax(300px, 1fr));
  gap: 24px;
}

Holy Grail Layout

.page {
  display: grid;
  grid-template:
    "header header header" 80px
    "nav    main   aside"  1fr
    "footer footer footer" 60px
    / 200px 1fr    200px;
  min-height: 100vh;
}

  5. Make the layout fill the viewport height with min-height: 100vh

Key Takeaways

  • Grid is for 2D layouts — rows AND columns together
  • fr units divide available space proportionally
  • repeat(auto-fit, minmax()) creates responsive grids automatically
  • Named areas make complex layouts readable
