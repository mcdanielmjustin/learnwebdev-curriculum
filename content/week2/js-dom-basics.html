What is the DOM?

The Document Object Model (DOM) is how JavaScript sees your HTML page. It's a tree-like structure where each HTML element becomes an object that JavaScript can access, modify, add, or delete.

üí° Think of it Like This

If HTML is the blueprint of a house, the DOM is the actual house after it's built. JavaScript is like a contractor who can walk through the house, repaint walls (change content), add furniture (add elements), or remodel rooms (modify structure).

HTML ‚Üí DOM Tree
<!-- This HTML... --> <div class="card"> <h2>Welcome</h2> <p>Hello, World!</p> </div> /* ...becomes this DOM tree: document ‚îî‚îÄ‚îÄ html ‚îî‚îÄ‚îÄ body ‚îî‚îÄ‚îÄ div.card ‚îú‚îÄ‚îÄ h2 ‚Üí "Welcome" ‚îî‚îÄ‚îÄ p ‚Üí "Hello, World!" */

Selecting Elements

Before you can change something on the page, you need to select it. Here are the main methods:

Method Returns Use When
getElementById() Single element or null You have a unique ID
querySelector() First matching element You want any CSS selector (recommended!)
querySelectorAll() NodeList (array-like) You need ALL matching elements
getElementsByClassName() HTMLCollection Legacy - prefer querySelector
JavaScript - Selecting Elements
// Given this HTML: // <h1 id="title">Welcome</h1> // <p class="intro">First paragraph</p> // <p class="intro">Second paragraph</p> // <button class="btn primary">Click Me</button> // getElementById - select by ID (most specific) const title = document.getElementById('title'); // querySelector - select FIRST match using CSS selector const firstIntro = document.querySelector('.intro'); const button = document.querySelector('.btn.primary'); const heading = document.querySelector('h1'); // querySelectorAll - select ALL matches const allIntros = document.querySelectorAll('.intro'); console.log(allIntros.length); // 2 // Complex selectors work too! const navLinks = document.querySelectorAll('nav a'); const firstCard = document.querySelector('.card:first-child');
‚úÖ Prefer querySelector

Use querySelector and querySelectorAll for most selections. They accept any CSS selector, making them extremely flexible.

Changing Content

Once you have an element, you can change its content:

JavaScript
const heading = document.querySelector('h1'); // textContent - for plain text (safe, recommended) heading.textContent = 'New Title'; // innerHTML - for HTML content (be careful with user input!) heading.innerHTML = 'Welcome <span>User</span>'; // Reading content console.log(heading.textContent); // "Welcome User" // Practical example: Update score display const scoreDisplay = document.querySelector('#score'); let score = 0; function updateScore(points) { score += points; scoreDisplay.textContent = `Score: ${score}`; }

Changing Styles

You can modify CSS directly or work with classes:

JavaScript - Inline Styles
const box = document.querySelector('.box'); // Set individual style properties box.style.backgroundColor = '#6366f1'; box.style.color = 'white'; box.style.padding = '20px'; box.style.borderRadius = '8px'; // Note: CSS properties become camelCase in JS // background-color ‚Üí backgroundColor // font-size ‚Üí fontSize // border-radius ‚Üí borderRadius
JavaScript - Working with Classes (Better!)
const button = document.querySelector('.btn'); // classList methods button.classList.add('active'); // Add a class button.classList.remove('active'); // Remove a class button.classList.toggle('active'); // Toggle on/off button.classList.contains('active'); // Returns true/false // Add multiple classes button.classList.add('active', 'highlighted'); // Practical: Toggle dark mode function toggleDarkMode() { document.body.classList.toggle('dark-mode'); }
‚úÖ Prefer Classes Over Inline Styles

Using classList keeps your styling in CSS where it belongs. This makes code more maintainable and enables CSS transitions/animations.

Changing Attributes

Modify HTML attributes like src, href, disabled, etc:

JavaScript
const link = document.querySelector('a'); const image = document.querySelector('img'); const input = document.querySelector('input'); // Get attribute value const currentHref = link.getAttribute('href'); // Set attribute value link.setAttribute('href', 'https://example.com'); link.setAttribute('target', '_blank'); // Common shortcuts for standard attributes image.src = 'new-image.jpg'; image.alt = 'New description'; link.href = 'https://example.com'; // Boolean attributes (disabled, checked, required) input.disabled = true; // Disable input input.disabled = false; // Enable input // Remove attribute link.removeAttribute('target'); // Data attributes (data-*) // <div data-question-id="42" data-category="ethics"> const card = document.querySelector('[data-question-id]'); const id = card.dataset.questionId; // "42" const category = card.dataset.category; // "ethics"

Creating & Removing Elements

Add new elements to the page or remove existing ones:

JavaScript - Creating Elements
// Create a new element const newCard = document.createElement('div'); // Set its properties newCard.className = 'card'; newCard.innerHTML = ` <h3>New Question</h3> <p>This card was created with JavaScript!</p> `; // Add it to the page const container = document.querySelector('.container'); container.appendChild(newCard); // Add at the end container.prepend(newCard); // Add at the beginning // Insert before a specific element const existingCard = document.querySelector('.card'); container.insertBefore(newCard, existingCard);
JavaScript - Removing Elements
// Remove an element const card = document.querySelector('.card'); card.remove(); // Clear all children const container = document.querySelector('.container'); container.innerHTML = ''; // Removes all content

Practical Example: Building a Simple Quiz Display

HTML
<div id="quiz-container"> <h2 id="question-text">Loading...</h2> <div id="options"></div> <p id="feedback"></p> <p>Score: <span id="score">0</span></p> </div>
JavaScript
// Quiz data const question = { text: 'What is the primary purpose of informed consent?', options: [ 'To protect the therapist legally', 'To ensure client autonomy and understanding', 'To increase billing rates', 'To document sessions' ], correct: 1 }; let score = 0; // Get DOM elements const questionText = document.querySelector('#question-text'); const optionsContainer = document.querySelector('#options'); const feedback = document.querySelector('#feedback'); const scoreDisplay = document.querySelector('#score'); // Display the question function displayQuestion() { questionText.textContent = question.text; optionsContainer.innerHTML = ''; // Clear previous question.options.forEach((option, index) => { const button = document.createElement('button'); button.textContent = option; button.className = 'option-btn'; button.dataset.index = index; button.addEventListener('click', checkAnswer); optionsContainer.appendChild(button); }); } // Check the answer function checkAnswer(e) { const selected = parseInt(e.target.dataset.index); if (selected === question.correct) { score++; feedback.textContent = '‚úì Correct!'; feedback.style.color = 'green'; e.target.classList.add('correct'); } else { feedback.textContent = '‚úó Incorrect'; feedback.style.color = 'red'; e.target.classList.add('incorrect'); } scoreDisplay.textContent = score; } // Start the quiz displayQuestion();

Key Takeaways

  • The DOM is JavaScript's representation of your HTML page
  • Use querySelector/querySelectorAll to select elements
  • Change content with textContent (safe) or innerHTML (for HTML)
  • Prefer classList methods over inline styles
  • Create elements with createElement, add them with appendChild
  • Remove elements with element.remove()
