Flexbox Layout

45 min Lesson 8 of 28
What you'll learn

Why Flexbox?

Before Flexbox, centering things in CSS was notoriously difficult. Floats were hacky, and vertical centering was a nightmare. Flexbox solves all of this with a powerful, intuitive layout system.

Flexbox is designed for one-dimensional layouts — either a row or a column. It excels at:

  • Distributing space between items
  • Aligning items horizontally and vertically
  • Creating responsive layouts without media queries
  • Reordering elements visually

The Basics: Container & Items

Flexbox has two key concepts:

  • Flex Container: The parent element with display: flex
  • Flex Items: The direct children of a flex container
flexbox-basics.css 
/* The parent becomes a flex container */
.container {
  display: flex;
}

/* Children automatically become flex items */
.item {
  /* No special CSS needed - they're flex items now! */
}

Main Axis vs Cross Axis

Flexbox works on two axes:

  • Main Axis: The primary direction (horizontal by default)
  • Cross Axis: Perpendicular to the main axis (vertical by default)
/* Change direction with flex-direction */
.row { flex-direction: row; }         /* default: left to right */
.row-reverse { flex-direction: row-reverse; }  /* right to left */
.column { flex-direction: column; }      /* top to bottom */
.column-reverse { flex-direction: column-reverse; } /* bottom to top */

justify-content: Main Axis Alignment

This controls how items are distributed along the main axis.

justify-content.css 
.container {
  display: flex;
  justify-content: flex-start;     /* default: pack items at start */
  justify-content: flex-end;       /* pack at end */
  justify-content: center;         /* center items */
  justify-content: space-between;  /* equal space between items */
  justify-content: space-around;   /* equal space around items */
  justify-content: space-evenly;   /* truly equal spacing */
}

align-items: Cross Axis Alignment

This controls alignment along the cross axis (vertical for row layouts).

.container {
  display: flex;
  align-items: stretch;     /* default: stretch to fill container */
  align-items: flex-start;  /* align to top */
  align-items: flex-end;    /* align to bottom */
  align-items: center;      /* center vertically */
}
align-items: center (perfect vertical centering!)
Perfectly Centered!
The Holy Grail of Centering

To center something both horizontally and vertically: display: flex; justify-content: center; align-items: center;. Three lines. That's it!

gap: Spacing Between Items

The gap property adds space between flex items (not around the edges).

.container {
  display: flex;
  gap: 16px;         /* Equal gap between all items */
  gap: 16px 32px;    /* row-gap | column-gap */
}

Real Example: Navigation Bar

nav.css 
.nav {
  display: flex;
  justify-content: space-between;  /* Logo left, links right */
  align-items: center;             /* Vertically center */
  padding: 16px 32px;
  background: #09090b;
}

.nav-links {
  display: flex;
  gap: 24px;                       /* Space between links */
}

.nav-links a {
  color: #a1a1aa;
  text-decoration: none;
}

.nav-links a:hover {
  color: #fafafa;
}

flex-wrap: Handling Overflow

By default, flex items try to fit on one line. Use flex-wrap to allow wrapping.

.container {
  display: flex;
  flex-wrap: nowrap;   /* default: squeeze onto one line */
  flex-wrap: wrap;     /* allow items to wrap to next line */
}

Key Takeaways

  • display: flex on the parent creates a flex container
  • justify-content aligns on the main axis (usually horizontal)
  • align-items aligns on the cross axis (usually vertical)
  • gap adds space between items
  • Centering is easy: justify-content: center; align-items: center;
