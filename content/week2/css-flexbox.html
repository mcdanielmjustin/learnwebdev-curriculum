Why Flexbox?

Before Flexbox, centering things in CSS was notoriously difficult. Floats were hacky, and vertical centering was a nightmare. Flexbox solves all of this with a powerful, intuitive layout system.

Flexbox is designed for one-dimensional layouts — either a row or a column. It excels at:

  • Distributing space between items
  • Aligning items horizontally and vertically
  • Creating responsive layouts without media queries
  • Reordering elements visually

The Basics: Container & Items

Flexbox has two key concepts:

  • Flex Container: The parent element with display: flex
  • Flex Items: The direct children of a flex container

The Holy Grail of Centering

To center something both horizontally and vertically: display: flex; justify-content: center; align-items: center;. Three lines. That's it!

Key Takeaways

  • display: flex on the parent creates a flex container
  • justify-content aligns on the main axis (usually horizontal)
  • align-items aligns on the cross axis (usually vertical)
  • gap adds space between items
  • Centering is easy: justify-content: center; align-items: center;
