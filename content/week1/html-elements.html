HTML Elements: Text & Lists

45 min Lesson 2 of 28
What you'll learn
  • Use headings (h1-h6) to structure content
  • Create paragraphs and line breaks
  • Build ordered and unordered lists

Headings: h1 through h6

Headings create hierarchy in your content. They range from <h1> (most important) to <h6> (least important).

headings.html 
<h1>Main Page Title</h1>
<h2>Section Heading</h2>
<h3>Subsection</h3>
<h4>Sub-subsection</h4>
<h5>Minor heading</h5>
<h6>Smallest heading</h6>
SEO Tip

Use only ONE <h1> per page. Search engines use it to understand your page's main topic. Use h2-h6 for subsections.

Text Elements

Paragraphs

The <p> tag defines a paragraph. Browsers automatically add space before and after paragraphs.

<p>This is a paragraph of text. It can be as long as you want.</p>
<p>This is another paragraph. Notice the automatic spacing.</p>

Text Formatting

Element Purpose Example
<strong> Important text (bold) Important
<em> Emphasized text (italic) Emphasized
<mark> Highlighted text Highlighted
<small> Smaller text Small print
<del> Deleted/strikethrough Removed 
<p>
  This text is <strong>very important</strong> and this is
  <em>emphasized</em>. You can also <mark>highlight</mark> text.
</p>

Lists

HTML has three types of lists. You'll use the first two constantly!

Unordered Lists (Bullets)

Use <ul> for lists where order doesn't matter:

<ul>
  <li>HTML</li>
  <li>CSS</li>
  <li>JavaScript</li>
</ul>

Ordered Lists (Numbers)

Use <ol> for sequential steps or rankings:

<ol>
  <li>Preheat oven to 350°F</li>
  <li>Mix ingredients</li>
  <li>Bake for 25 minutes</li>
</ol>

Nested Lists

You can put lists inside lists for subcategories:

<ul>
  <li>Frontend
    <ul>
      <li>HTML</li>
      <li>CSS</li>
      <li>JavaScript</li>
    </ul>
  </li>
  <li>Backend
    <ul>
      <li>Node.js</li>
      <li>Python</li>
    </ul>
  </li>
</ul>
Today's Project: Recipe Page

Create a recipe page using everything you've learned:

  1. Create recipe.html with proper document structure
  2. Add an <h1> with the recipe name
  3. Add a <p> describing the dish
  4. Create an <h2> for "Ingredients" followed by a <ul>
  5. Create an <h2> for "Instructions" followed by an <ol>
  6. Use <strong> to highlight important ingredients

Key Takeaways

  • Headings create hierarchy — h1 is most important, h6 least
  • Use semantic tags — <strong> not just bold for importance
  • ul for bullets, ol for numbers — both use <li> for items
