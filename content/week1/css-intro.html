What is CSS?

CSS stands for Cascading Style Sheets. While HTML defines the structure and content of a page, CSS controls how it looks — colors, fonts, spacing, layout, and more.

Think of it this way: HTML is the skeleton, CSS is the skin, clothes, and makeup.

Why "Cascading"?

Styles "cascade" down — if you style a parent element, children inherit those styles. Also, later rules can override earlier ones. This cascade is powerful once you understand it!

CSS Syntax

CSS uses a simple pattern: selector { property: value; }

style.css 
/* This is a CSS comment */

h1 {
  color: blue;
  font-size: 32px;
}

p {
  color: #333333;
  line-height: 1.6;
}
  • Selector: What element(s) to style (h1, p, .classname)
  • Property: What aspect to change (color, font-size)
  • Value: The new value (blue, 32px)
  • Semicolon: Each declaration ends with ;

Three Ways to Add CSS

1. External Stylesheet (Best Practice!)

Create a separate .css file and link it in your HTML:

index.html 
<head>
  <link rel="stylesheet" href="styles.css">
</head>

2. Internal Styles (In the <head>)

<head>
  <style>
    h1 { color: blue; }
  </style>
</head>

3. Inline Styles (Avoid When Possible)

<h1 style="color: blue; font-size: 32px;">Hello</h1>
Always Use External Stylesheets

External CSS files keep your code organized, can be cached by browsers (faster loading), and let you style multiple pages with one file. Only use inline styles for quick testing.

Selectors

Selectors determine which elements your styles apply to. Here are the most important ones:

Element Selector

Selects all elements of that type:

p { color: gray; }       /* All paragraphs */
h1 { font-size: 2rem; }  /* All h1 headings */

Class Selector

Selects elements with a specific class (starts with a dot):

/* CSS */
.highlight { background: yellow; }
.card { border: 1px solid #ddd; }

<!-- HTML -->
<p class="highlight">This is highlighted</p>
<div class="card">This is a card</div>

ID Selector

Selects ONE element with that ID (starts with #):

/* CSS */
#main-header { background: navy; }

<!-- HTML -->
<header id="main-header">...</header>
Classes vs IDs

Classes can be used multiple times on a page. IDs must be unique — only one element per ID. In practice, use classes for styling and IDs for JavaScript or anchor links.

Colors in CSS

CSS offers several ways to specify colors:

/* Named colors */
color: red;
color: dodgerblue;

/* Hex (most common) */
color: #ff0000;      /* red */
color: #3b82f6;      /* blue */

/* RGB */
color: rgb(255, 0, 0);

/* RGBA (with transparency) */
color: rgba(0, 0, 0, 0.5);  /* 50% transparent black */

Basic Text Styling

typography.css 
body {
  /* Font family with fallbacks */
  font-family: 'Inter', Arial, sans-serif;

  /* Base font size */
  font-size: 16px;

  /* Line spacing */
  line-height: 1.6;

  /* Text color */
  color: #333;
}

h1 {
  font-size: 2.5rem;      /* Relative to root font size */
  font-weight: 700;       /* Bold */
  text-align: center;    /* Center text */
  text-transform: uppercase;
}

a {
  color: #3b82f6;
  text-decoration: none;  /* Remove underline */
}

a:hover {
  text-decoration: underline;  /* Underline on hover */
}
Today's Project: Style Your About Me Page
  1. Create styles.css and link it to your about.html
  2. Set a nice font-family on the body (try: 'Segoe UI', Arial, sans-serif)
  3. Style your h1 with a custom color and larger size
  4. Create a .highlight class with a background color and apply it to important text
  5. Style links to be a different color, and add a hover effect
  6. Set a subtle background color on the body (#f5f5f5 works great)

Key Takeaways

  • CSS syntax: selector { property: value; }
  • Use external stylesheets — cleaner and more maintainable
  • Classes (.name) for reusable styles, IDs (#name) for unique elements
  • Colors: named, hex (#fff), rgb(), or rgba()
