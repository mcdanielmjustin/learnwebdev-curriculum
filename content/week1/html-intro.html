What is HTML? Document Structure

45 min Lesson 1 of 28
What you'll learn
  • Understand what HTML is and why it matters

  • Learn the basic document structure (DOCTYPE, html, head, body)

What is HTML?

HTML stands for HyperText Markup Language. It's the standard language for creating web pages. Every website you've ever visited is built with HTML at its core.

Think of HTML as the skeleton of a webpage. Just like your skeleton gives your body structure, HTML gives structure to web content. It tells the browser: "This is a heading," "This is a paragraph," "This is an image."

Key Insight

HTML doesn't make things look pretty — that's CSS's job. HTML defines what content is (a heading, a list, a link). CSS defines how it looks (colors, fonts, spacing).

Your First HTML File

Let's create your very first HTML file. You'll be amazed how simple it is to get started!

Step 1: Create a file

Create a new file called index.html on your computer. You can use any text editor — VS Code, Sublime Text, or even Notepad.

Pro Tip

We recommend VS Code (Visual Studio Code). It's free, powerful, and has excellent HTML support. Download it at code.visualstudio.com

Step 2: Write your first HTML

Type this code into your file:

index.html 
<!-- This is your first HTML file! -->
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <title>My First Webpage</title>
</head>
<body>
  <h1>Hello, World!</h1>
  <p>I just wrote my first HTML!</p>
</body>
</html>

Step 3: Open in your browser

Save the file, then double-click it to open in your web browser. You should see your heading and paragraph displayed!

Understanding the Structure

Let's break down each part of this HTML document:

<!DOCTYPE html>

This declaration tells the browser "Hey, this is an HTML5 document!" It must be the very first line.

<html>

The root element that contains ALL other HTML elements. The lang="en" attribute tells browsers and search engines this page is in English.

<head>

The "brain" of your document. Contains metadata — information ABOUT your page that doesn't appear on the page itself:

  • <meta charset="UTF-8"> — Character encoding (supports all languages)
  • <title> — Shows in the browser tab
  • CSS stylesheets and JavaScript files also go here

<body>

Everything visible on your webpage goes inside the body. This is where all your content lives — text, images, buttons, everything!

Common Mistake

Every opening tag needs a closing tag! <h1> needs </h1>. Forgetting the slash (/) is one of the most common beginner errors.

Tags, Elements & Attributes

Three key terms you'll use constantly:

Tags

Tags are the building blocks of HTML. They're wrapped in angle brackets: <tagname>

Most tags come in pairs: an opening tag and a closing tag (with a slash).

Elements

An element is the complete package: opening tag + content + closing tag.

<p>This entire thing is a paragraph element</p>

Attributes

Attributes provide extra information about elements. They go inside the opening tag:

<a href="https://google.com">Click me!</a>

<!-- href is an attribute that tells the link where to go -->
Try It Yourself!
  1. Create a new file called about.html
  2. Copy the basic HTML structure from above
  3. Change the <title> to "About Me"
  4. In the body, add:
    • An <h1> with your name
    • A <p> describing yourself
    • Another <p> about why you want to learn web development
  5. Open it in your browser and admire your work!

Key Takeaways

  • HTML is structure — it defines what content IS, not how it looks
  • Every HTML page needs: DOCTYPE, html, head, and body
  • Tags come in pairs — opening <tag> and closing </tag>
  • Attributes add extra info to elements

Congratulations! You've written your first HTML. Tomorrow, we'll learn about more HTML elements like headings, lists, and text formatting. See you then!
