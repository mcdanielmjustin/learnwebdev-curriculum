The CSS Box Model

45 min Lesson 5 of 28
What you'll learn
  • Understand the box model: content, padding, border, margin
  • Control element sizing with width and height
  • Use box-sizing: border-box for predictable layouts

Everything is a Box

In CSS, every element is a rectangular box. Even circles and images — they all occupy rectangular space. Understanding the box model is fundamental to CSS layout.

Every box has four layers, from inside to out:

Margin
Border
Padding
Content
  • Content — The actual text, image, or child elements
  • Padding — Space between content and border (inside the box)
  • Border — The visible edge of the box
  • Margin — Space outside the border (between this box and others)

Padding

Padding creates space inside the element, between the content and the border. It takes on the element's background color.

padding.css 
/* All four sides */
.box {
  padding: 20px;
}

/* Vertical | Horizontal */
.box {
  padding: 10px 20px;  /* 10px top/bottom, 20px left/right */
}

/* Top | Horizontal | Bottom */
.box {
  padding: 10px 20px 30px;
}

/* Top | Right | Bottom | Left (clockwise) */
.box {
  padding: 10px 20px 30px 40px;
}

/* Individual sides */
.box {
  padding-top: 10px;
  padding-right: 20px;
  padding-bottom: 10px;
  padding-left: 20px;
}

Margin

Margin creates space outside the element. It's transparent and pushes other elements away.

/* Same shorthand patterns as padding */
.box {
  margin: 20px;           /* All sides */
  margin: 20px 40px;      /* Vertical | Horizontal */
  margin: 10px 20px 30px 40px;  /* Clockwise */
}

/* Center a block element horizontally */
.centered {
  width: 600px;
  margin: 0 auto;  /* auto calculates equal left/right margins */
}
Margin Collapse

When two vertical margins touch, they "collapse" into one (the larger value wins). This only happens vertically, not horizontally. It's a common source of confusion!

Border

Borders are visible edges around elements. You can control width, style, and color.

borders.css 
/* Shorthand: width style color */
.box {
  border: 2px solid #3b82f6;
}

/* Individual properties */
.box {
  border-width: 2px;
  border-style: solid;   /* solid, dashed, dotted, double, none */
  border-color: #3b82f6;
}

/* Single side */
.box {
  border-bottom: 3px solid #22c55e;
}

/* Rounded corners */
.box {
  border-radius: 8px;      /* All corners */
  border-radius: 50%;      /* Circle (on square elements) */
  border-radius: 8px 0 0 8px;  /* Top-left, top-right, bottom-right, bottom-left */
}

Width and Height

By default, width and height only set the content size. Padding and border are added on top!

.box {
  width: 300px;
  padding: 20px;
  border: 5px solid black;
}
/* Total width = 300 + 20 + 20 + 5 + 5 = 350px! */
The Math Problem

With default box-sizing, a 300px wide box with 20px padding and 5px border is actually 350px wide. This makes layouts frustrating!

The Fix: box-sizing: border-box

This CSS property changes how width/height work. With border-box, padding and border are included in the width.

reset.css 
/* Apply to ALL elements - do this in every project! */
* {
  box-sizing: border-box;
}

/* Now this box is exactly 300px wide total */
.box {
  width: 300px;
  padding: 20px;
  border: 5px solid black;
}
/* Content area shrinks to fit: 300 - 40 - 10 = 250px */
Always Use border-box

Add * { box-sizing: border-box; } at the top of every stylesheet. This is standard practice and makes layouts much more predictable.

Practical Example: Card Component

card.css 
.card {
  width: 320px;
  padding: 24px;
  margin: 16px;

  background: #1c1c1f;
  border: 1px solid rgba(255, 255, 255, 0.1);
  border-radius: 12px;
}

.card-title {
  margin-bottom: 12px;
  padding-bottom: 12px;
  border-bottom: 1px solid rgba(255, 255, 255, 0.1);
}

.card-content {
  padding: 0;  /* Content doesn't need extra padding - card has it */
}
Today's Project: Styled Cards
  1. Add * { box-sizing: border-box; } to your stylesheet
  2. Create a .card class with:
    • Fixed width (300-350px)
    • Padding (20-24px)
    • Border and border-radius
    • Background color
  3. Create 3 cards in your HTML, each with a title and description
  4. Use margin to space them apart
  5. Experiment: What happens when you change padding? Border-radius?

Key Takeaways

  • Four layers: Content → Padding → Border → Margin
  • Padding is inside, margin is outside
  • Always use box-sizing: border-box for predictable sizing
  • Shorthand: padding: top right bottom left (clockwise)
