Part 2: Final Project

üöÄ Build a Landing Page from Scratch

Put everything together! Build a complete landing page using only HTML and CSS. This is your chance to apply all the concepts from Week 1.

Requirements Checklist

  • Proper HTML5 document structure (DOCTYPE, html, head, body)
  • Semantic HTML tags (header, nav, main, section, footer)
  • A fixed navigation header with logo and links
  • A hero section with heading, paragraph, and button
  • At least one image with proper alt text
  • A features or cards section (3+ items)
  • External CSS file (styles.css) linked properly
  • box-sizing: border-box applied globally
  • Consistent spacing using padding and margin
  • At least 2 hover effects (links, buttons)

Starter Template

Start with this HTML structure and build from there:

index.html 
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>My Landing Page</title>
  <link rel="stylesheet" href="styles.css">
</head>
<body>
  <header>
    <nav>
      <div class="logo">MyBrand</div>
      <div class="nav-links">
        <a href="#features">Features</a>
        <a href="#about">About</a>
        <a href="#contact">Contact</a>
      </div>
    </nav>
  </header>

  <main>
    <section class="hero">
      <h1>Welcome to MyBrand</h1>
      <p>Your tagline goes here. Make it compelling!</p>
      <a href="#" class="btn">Get Started</a>
    </section>

    <section id="features" class="features">
      <h2>Features</h2>
      <div class="cards">
        <!-- Add your feature cards here -->
      </div>
    </section>
  </main>

  <footer>
    <p>&copy; 2026 MyBrand. All rights reserved.</p>
  </footer>
</body>
</html>

Tips for Success

  • Start with structure: Get all your HTML in place before styling
  • Work section by section: Style the header, then hero, then features
  • Use your browser's DevTools: Right-click ‚Üí Inspect to debug
  • Reference the PassEPPP site: Look at how it's structured for inspiration
  • Don't overthink it: Simple and clean beats complex and messy
