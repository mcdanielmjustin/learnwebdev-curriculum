# LearnWebDev - 28-Day Web Development Curriculum

A complete curriculum for learning web development from scratch. Build a real web application while mastering HTML, CSS, and JavaScript.

## How to Use

Clone this repo and open `index.html` in your browser, or deploy to any static hosting service.

## Curriculum Overview

### Week 1: HTML & CSS Fundamentals (7 lessons)
- HTML document structure, elements, links, images
- CSS selectors, box model, layout basics

### Week 2: Advanced CSS & JavaScript Basics (7 lessons)  
- Flexbox, Grid, responsive design
- JavaScript variables, functions, DOM manipulation

### Week 3: JavaScript Advanced & Components (7 lessons)
- Events, arrays, objects, async/await, fetch API
- Building navigation and modal components

### Week 4: Full-Stack Development (7 lessons)
- Auth0 authentication, Supabase database
- Building landing page, dashboard, practice page, calendar
- Final project and deployment

## Local Development

All files are in the local `PassEPPP-dev-learning` directory. To push to this repo:

```bash
git remote add origin https://github.com/mcdanielmjustin/learnwebdev-curriculum.git
git push -u origin main
```
